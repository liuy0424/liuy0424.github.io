SQL 基础 (02)

2020-01-10数据库

查询基础

列的查询

SELECT 语句的语法

SELECT <列名>,...
    FROM <表名>;

示例：从 Product 表中输出3列

SELECT product_id, product_name, purchase_price
    FROM product;

查询出表中的所有列

语法：

SELECT *
    FROM <表名>;

示例：

SELECT *
    FROM product;

为列设定别名

SQL语句使用 AS 关键字为列设定别名。

示例：

SELECT product_id AS id,
       product_name AS name,
       purchase_price AS price
    FROM product;

示例2：设定中文别名

SELECT product_id AS "商品编号",
       product_name AS "商品名称",
       purchase_price AS "进货单价"
    FROM product;

注： 使用中文别名要用双引号

常数的查询

示例：

SELECT '商品' AS string, 38 AS number, '2009-02-24' AS date,
       product_id, product_name
    FROM product;

从结果中删除重复行

示例： 使用DISTINCT删除重复数据

SELECT DISTINCT product_type
    FROM product;

DISTINCT 关键字只能用在第一个列名之前。

根据 WHERE 语句来选择记录

语法：

SELECT <列名>,...
    FROM <表名>
    WHERE <条件表达式>;

示例：选取 product_type 为 衣服 的记录

SELECT DISTINCT product_name, product_type
    FROM product
    WHERE product_type = '衣服';

注释的写法

  • 单行注释： --
  • 多行注释： /* … */

算术运算符

示例：将 sale_price的价格乘以2.

SELECT product_name, sale_price,
       sale_price * 2 AS "sale_proce *2"
    FROM product;

运算是以 行 为单位执行的

SQL的算术运算符包括：

运算符 含义
+ 加法
- 减法
* 乘法
/ 除法

所有包含NULL 的计算，结果都是 NULL

比较运算符

SQL 比较运算符包括：

运算符 含义
= 等于
<> 不等于
>= 大于等于
> 大于
<= 小于等于
< 小于

示例：选取出销售单价大于等于1000的记录

SELECT product_name, sale_price,sale_price
    FROM product
    WHERE sale_price >= 1000;

示例：选取出登记日期在2009年9月27日之前的记录

SELECT product_name, sale_price,regist_date
    FROM product
    WHERE regist_date < '2009-9-27';

示例： WHERE 子句的条件表达式也可以使用计算表达式

-- 选出sale_price比 purchase_price高出500元的商品
SELECT product_name, sale_price,purchase_price
    FROM product
    WHERE sale_price - purchase_price >= 500;

注：
字符串类型的数据原则上按照字符顺序进行排序，与数字的大小顺序不同。

选取NULL 的记录

示例： 使用 IS NULL 关键字来选取 NULL 数据

SELECT product_name, purchase_price
    FROM product
    WHERE purchase_price IS NULL;

示例： 选取不是NULL 的 数据

SELECT product_name, purchase_price
    FROM product
    WHERE purchase_price IS NOT NULL;

小结：
希望选取 NULL 记录，在条件表达式中使用 IS NULL 运算符
希望选取不是 NULL 记录，在条件表达式中使用 IS NOT NULL 运算符


逻辑运算符

NOT 运算符

示例： 选取单价小于1000元的商品

SELECT product_name, product_type, sale_price
    FROM product
    WHERE NOT  sale_price >= 1000;

注： NOT 运算符用来否定某一条件

AND 和 OR 运算符

AND运算符: 所有条件都成立时为真。
OR运算符: 条件中只要有一个为真时即为真。
示例：同时满足商品类型是”厨房用具”并且价格高于或等于3000的商品。

SELECT product_name, purchase_price
    FROM product
    WHERE product_type = '厨房用具'
        AND sale_price >= 3000;  -- 两个条件的交集

示例：满足商品类型是”厨房用具”或者价格高于或等于3000的商品。

SELECT product_name, purchase_price
    FROM product
    WHERE product_type = '厨房用具'
        OR sale_price >= 3000;  -- 两个条件的并集

通过括号改变优先级

示例：
满足下列条件的记录 –
“商品种类为办公用品”
并且 “登记日期是2009年9月11日或者2009年9月20日”

SELECT product_name, product_type, regist_date
    FROM product
    WHERE product_type = '办公用品'
        AND ( regist_date = '2009-09-11'
        OR regist_date = '2009-09-20');

注： AND 运算符高于 OR

含有 NULL 时的真值

正常来说，NULL 不应参与逻辑运算。因为NULL 即不是True, 也不是False。

如果NULL参与了运算，那么规则如下：

AND:

逻辑值1 逻辑值2 AND 结果
True NULL NULL
False NULL False
NULL True NULL
NULL False False

OR:

逻辑值1 逻辑值2 OR 结果
True NULL True
False NULL NULL
NULL True True
NULL False NULL
