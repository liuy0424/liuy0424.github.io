各种各亲的函数

函数的种类

函数：就是输入某一值得到相应输出结果的功能。
输入值称为参数, 输出值称为返回值。

函数包括以下几类：

  • 算术函数
  • 字符串函数
  • 日期函数
  • 转换函数
  • 聚合函数

算术函数

算术函数包括：

  • +、-、*、/
  • ABS 函数 – 绝对值
  • MOD 函数 – 求余
  • ROUND 函数 – 四舍五入

示例：

-- ABS
SELECT m, ABS(m) as abs_col
FROM SAMPLEMATH;

-- MOD (SQL Server不支持这个函数，用 % 来代替)
SELECT n,p, mod(n,p) AS mod_col
FROM SAMPLEMATH;
-- ROUND
SELECT m,n,
       round(m,n) AS round_col
FROM SAMPLEMATH;

大多数函数 NULL的返回值都是NULL.

字符串函数

字符串函数包括：

  • || 函数 – 拼接字符串
  • LENGTH 函数 – 字符串长度
  • LOWER 函数 – 小写转换
  • REPLACE 函数 – 字符串的替换
  • SUBSTRING – 字符串的截取
  • UPPER – 大写转换

示例：

-- || 函数 （在SQL server 和 MySQL 中不支持）
SELECT STR1, STR2,
       STR1 || STR2 AS str_concat
FROM SAMPLESTR;

-- 三个字符串拼接
SELECT STR1, STR2, STR3,
       STR1 || STR2 || STR3 AS str_concat
FROM SAMPLESTR;

-- LENGTH (无法在SQL Server中使用，用 LEN 代替)
SELECT STR1,
       length(STR1) AS len_str
FROM SAMPLESTR;

-- LOWER
SELECT STR1,
       lower(STR1) AS low_str
FROM SAMPLESTR;

-- REPLACE
SELECT STR1, STR2, STR3,
       replace(str1, str2,str3) AS rep_str
FROM SAMPLESTR;

-- SUBSTRING
SELECT STR1,
       substring(STR1 FROM 3 FOR 2) AS sub_str
FROM SAMPLESTR;

-- UPPER
SELECT STR1,
       upper(STR1) AS up_str
FROM SAMPLESTR
WHERE str1 IN ('ABC', 'aBC', 'abc', '山田');

日期函数

日期函数包括：

  • CURRENT_DATE 函数 – 当前函数
  • CURRENT_TIME – 当前时间
  • CURRENT_TIMESTAMP – 当前日期和时间
  • EXTRACT – 截取日期元素

示例：

-- CURRENT_DATE
SELECT current_date;

-- CURRENT_TIME
SELECT current_time;

-- CURRENT_TIMESTAMP
SELECT current_timestamp;

-- EXTRACT
SELECT CURRENT_TIMESTAMP,
EXTRACT(YEAR FROM CURRENT_TIMESTAMP) AS year,
EXTRACT(MONTH FROM CURRENT_TIMESTAMP) AS month,
EXTRACT(DAY FROM CURRENT_TIMESTAMP) AS day,
EXTRACT(HOUR FROM CURRENT_TIMESTAMP) AS hour,
EXTRACT(MINUTE FROM CURRENT_TIMESTAMP) AS minute,
EXTRACT(SECOND FROM CURRENT_TIMESTAMP) AS second;

转换函数

转换函数包括：

  • CAST – 类型转换
  • COALESCE – 将 NULL 转换成其它值

示例：

-- CAST
-- 字符串类型转换成数值
SELECT CAST('0001' AS INTEGER) AS int_col;

-- 字符串类型转换为日期类型
SELECT CAST('2009-12-14' AS DATE) AS date_col;

-- COALESCE 
-- 返回第一个非NULL 的值
SELECT COALESCE(NULL, 1) AS col_1,
        COALESCE(NULL, 'test', NULL) AS col_2,
        COALESCE(NULL, NULL, '2009-11-01') AS col_3;

-- 将返回NULL 的值显示成字符串"NULL"
SELECT COALESCE(str2, 'NULL')
    FROM SAMPLESTR;

谓词

什么是谓词

谓词就是用于布尔值判断的一些关键词。返回值为 True/False/Unknown
谓词包括：

  • LIKE – 部分匹配
  • BETWEEN – 范围查询
  • IS NULL、 IS NOT NULL – 判断是否为NULL.
  • IN – 判断是否在集合范围内，是 OR 的简便用法
  • EXISTS

LIKE 谓词

比较是否字符串部分一致。
部分一致包括：前方一致、中间一致、后方一致
示例：

-- 前方一致
SELECT * FROM SAMPLELIKE
    WHERE STRCOL LIKE 'ddd%';

-- 中间一致
SELECT * FROM SAMPLELIKE
    WHERE STRCOL LIKE '%ddd%';

-- 后方一致
SELECT * FROM SAMPLELIKE
    WHERE STRCOL LIKE '%ddd';

-- 使用_ 来匹配
SELECT * FROM SAMPLELIKE
    WHERE STRCOL LIKE 'abc__';      -- 匹配 abcdd

-- 使用_ 来匹配
SELECT * FROM SAMPLELIKE
    WHERE STRCOL LIKE 'abc___';      -- 匹配 abcddd

LIKE 匹配时， % 匹配一个或多个字符， _ 匹配单个字符。

BETWEEN 谓词

BETWEEN 用于查询范围
示例： 选取销售单价为100~1000元的商品

SELECT product_name, sale_price
FROM PRODUCT
WHERE SALE_PRICE BETWEEN 100 AND 1000;

BETWEEN 的特点是结果中会包含两边的临界值。

IS NULL、IS NOT NULL 谓词

示例：

SELECT product_name, PURCHASE_PRICE
FROM PRODUCT
WHERE PURCHASE_PRICE IS NULL ;

SELECT product_name, PURCHASE_PRICE
FROM PRODUCT
WHERE PURCHASE_PRICE IS NOT NULL ;

IN 谓词

代替 OR 来简化 SQL 写法
示例：

-- 通过IN 来指定多个进货单价进行查询
SELECT product_name, PURCHASE_PRICE
FROM PRODUCT
WHERE PURCHASE_PRICE IN (320, 500, 5000);

-- 通过 NOT IN 来排除多个进货单价进行查询
SELECT product_name, PURCHASE_PRICE
FROM PRODUCT
WHERE PURCHASE_PRICE NOT IN (320, 500, 5000);

如果想要排除某个范围，可以使用 NOT 来实现。
无论 IN 或者 NOT IN， NULL 值都不会被选取到。

使用子查询作为IN谓词的参数

示例：取得“在大阪店销售的商品的销售单价”

SELECT product_name, sale_price
FROM PRODUCT
WHERE PRODUCT_ID IN (
    SELECT product_id
    FROM SHOPPRODUCT
    WHERE SHOP_ID = '000C'
    );

这种方式的好处是，IN 的范围可以是动态变化的。

示例： NOT IN 

FROM PRODUCT
WHERE PRODUCT_ID NOT IN (
    SELECT product_id
    FROM SHOPPRODUCT
    WHERE SHOP_ID = '000C'
    );

这会返回除了0001、0002、0003之外的结果

EXIST 谓词

示例：选取出大阪店（000C）在售商品（product_id）的销售单价(sale_price)

SELECT product_name, sale_price
FROM PRODUCT as P
WHERE exists(SELECT *
            FROM SHOPPRODUCT AS SP
            WHERE sp.SHOP_ID = '000C'
            AND sp.PRODUCT_ID = p.PRODUCT_ID);

EXISTS 是只有一个参数的谓词。该参数通常是一个关联子查询。
作为EXIST 参数的子查询中经常会使用 SELECT *。
示例： NOT EXIST

SELECT product_name, sale_price
FROM PRODUCT as P
WHERE NOT exists(SELECT *
            FROM SHOPPRODUCT AS SP
            WHERE sp.SHOP_ID = '000A'
            AND sp.PRODUCT_ID = p.PRODUCT_ID);

EXIST/NOT EXIST 通常可以用 IN/NOT IN 来代替

CASE 表达式

什么是CASE

用于处理条件分支。

CASE 表达式语法

语法：

CASE WHEN <求值表达式> THEN <表达式>
    WHEN <求值表达式> THEN <表达式>
    WHEN <求值表达式> THEN <表达式>
    ...
    ELSE <表达式>
END

CASE 表达式的使用方法

示例：通过CASE表达式将A ～ C的字符串加入到商品种类当中.

SELECT product_name,
       CASE WHEN product_type = '衣服' THEN 'A: ' || product_type
            WHEN product_type = '办公用品' THEN 'B: ' || product_type
            WHEN product_type = '厨房用具' THEN 'C: ' || product_type
            ELSE NULL
        END AS abc_product_type
FROM PRODUCT;

虽然 CASE 表达式中的 ELSE 子句可以省略，但建议不要省略。
CASE 表达式中的 END 不能省略

示例：使用CASE表达式进行行列转换

SELECT sum(CASE WHEN product_type = '衣服' THEN sale_price ELSE 0 END ) as sum_price_clothes,
       sum(CASE WHEN product_type = '厨房用具' THEN sale_price ELSE 0 END ) as sum_price_kitchen,
       sum(CASE WHEN product_type = '办公用品' THEN sale_price ELSE 0 END ) as sum_price_office
FROM PRODUCT;
